# Bundesliga API app

Bundesliga scores and results.

# Deployment

- git clone https://github.com/stanpet/mz3-app - clone the project
- composer install - install all dependencies
- php bin/console server:run - start the built-in Symfony web server

# Built With

Symfony 4.4
Bootstrap
Twig
